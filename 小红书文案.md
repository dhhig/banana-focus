# 🍌 剥 香 蕉

**一句话说清楚：勾选学习 App → 开始倒计时 → 切屏立刻弹窗 → 不切回来就作废。**

---

## 🎯 怎么逼你学习

```
① 勾选学习 App（比如 Notion + PDF Expert）
② 拖滑块选时长（5分钟 ~ 4小时自由定）
③ 点「开始专注」
④ 学习 App 自动打开，剥香蕉窗口淡出隐身
⑤ 菜单栏显示 🍌 38:42 实时倒计时
⑥ 你一切换到微信/B站/淘宝...
⑦ 💥 立刻弹窗：
   ┌──────────────────────┐
   │       🍌             │
   │   检测到切屏！        │
   │                      │
   │ [放弃学习] [继续学习] │
   └──────────────────────┘
⑧ 点「继续学习」→ 自动切回学习App，继续计时
   点「放弃学习」→ 计时作废，静默后台
⑨ 倒计时结束 → 自动完成 🎉
```

**没有借口。0.5秒检测。切屏就被抓。**

---

## ✨ 功能亮点

| 功能 | 说明 |
|------|------|
| 🎚️ **自由时长** | 滑块 5 分钟到 4 小时，不设固定档位 |
| 👀 **0.5秒轮询** | 双重检测机制，切屏瞬间反应 |
| 🪟 **智能弹窗** | 切屏弹出选项：继续 or 放弃 |
| 🔄 **自动切回** | 选「继续」自动回到学习App |
| 📊 **24h时间线** | 每小时一行，学习时段绿色高亮 |
| 📅 **月视图日历** | 达标那天显示 🍌，没达成 ⏳ |
| 🔥 **连续打卡** | Duolingo 式 streak，断了心疼 |
| 🎉 **达标庆祝** | 每天学满目标 → 弹窗 + 名言鼓励 |
| 🖥️ **状态栏倒计时** | 顶部显示 🍌 实时剩余时间 |
| 🍅 **番茄钟** | 25+5 自动循环，到点提醒 |

---

## 💡 为什么叫「剥香蕉」

香蕉 = 知识，剥香蕉 = 学习。

**日历上每达标一天就多一个 🍌。** 一个月 30 天有 25 个 🍌，你就是卷王。
