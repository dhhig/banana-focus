## 你是不是也这样 😭

用电脑学习的时候：
> 打开 Notion 准备写笔记 → 查资料打开了 Safari → 顺便看了眼微博 → 微信弹消息又点进去 → 回完消息又刷了会 B 站 → ...

**等回过神来，一小时过去了，一个字没写。**

我受够了。于是怒写代码，自己做了一个 macOS App，名字就叫——

# 🍌 剥香蕉

一句概括：**勾选学习App → 开始倒计时 → 一切屏就弹出警告 → 再切不回来就作废。简单粗暴。**

---

## 🎯 它怎么逼你学习

```
① 勾选你的学习App（比如 Notion + PDF Expert）
② 设置学习时长（5分钟~4小时自由滑块）
③ 点击"开始专注"
④ 学习App自动打开，剥香蕉窗口淡出隐身
⑤ 菜单栏显示 🍌 38:42 实时倒计时
⑥ 一切换到微信/B站/淘宝... 
⑦ 💥 立刻弹出浮窗：「检测到切屏！」
   [继续学习]  → 自动切回学习App，继续计时
   [放弃学习]  → 计时作废，记录失败
⑧ 倒计时结束 → 自动完成 + 庆祝动画
```

**没有借口。切屏就被抓。想继续就得乖乖切回去。**

---

## ✨ 我设计了这些细节

| 功能 | 体验 |
|------|------|
| 🎚️ **自由时长** | 滑块从5分钟拖到4小时，想学多久拖多久 |
| 📊 **24h时间线** | 每小时一行，学习时段绿色高亮，摸鱼时段灰色 |
| 📅 **月视图日历** | 达成目标那天显示🍌，没达成 ⏳，没学显示 · |
| 🍅 **番茄钟** | 25+5自动循环，到点提醒 |
| 🔥 **连续打卡** | 像 Duolingo 的 streak，断了心痛 |
| 🎉 **达标庆祝** | 每天学满目标 → 🍌弹跳+名言弹出 |
| 🖥️ **状态栏** | 顶部显示 🍌 倒计时，随时看剩余时间 |
| 🚀 **自动启动** | 勾选的学习App自动打开，不用手动找 |

---

## 💡 为什么要叫「剥香蕉」

香蕉=知识，剥香蕉=学习。每完成一次学习，就是在「剥一根香蕉」。

**月视图的日历上，学满目标的那天会出现一个 🍌。** 一个月有20个🍌，那个月就是赢麻了。
