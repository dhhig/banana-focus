# 🍌 我真的做了一个"剥香蕉"App来逼自己学习！！

---

## 你有没有过这种经历 😭

打开电脑准备学习 → 点开 Safari 查资料 → 不知不觉刷了半小时 B 站 → 又看了一下微信 → 又打开了淘宝 → ...... → 天黑了，一个字没写。

**我被自己气死！！！**

于是怒学 SwiftUI，花了几个晚上，自己做了一个 macOS 专注学习 App，名字就叫——

# 🍌 剥 香 蕉

---

## 它的逻辑非常简单粗暴 👇

```
1️⃣ 勾选你学习要用的 App（比如 Notion + PDF Expert）
2️⃣ 点击「开始专注」
3️⃣ 屏幕自动暗化，只有你选的 App 窗口是亮的 ✨
4️⃣ 一切换到别的 App → 5秒宽限期 ⚠️
5️⃣ 超时不切回来 → 本次学习直接作废 ❌
6️⃣ 完成学习 → 时长自动记录 📊
7️⃣ 每天达标 → 🍌 香蕉剥完！弹出庆祝动画 🎉
```

**切屏就断蕉！没有借口！**

---

## 我做了哪些功能

| 功能 | 说人话 |
|------|--------|
| 🌑 **屏幕暗化** | 非学习区全黑，强迫你只看学习App |
| ⏱️ **24h 时间线** | 学了多久、几点学的，一目了然 |
| 📅 **月视图日历** | 每天达标就显示 🍌，整月打卡超有成就感 |
| 🍅 **番茄钟** | 25分钟学+5分钟休，内置提醒 |
| 🔥 **连续打卡** | 像 Duolingo 一样的 streak，断了心痛 |
| 🎉 **达标动画** | 香蕉弹跳 + 名人名言，学完有仪式感 |
| 📊 **年视图** | 12 个月学习时长柱状图，年度复盘利器 |

---

## 我设计的一些小心思 💡

- **Icon 是剥了一半的香蕉** 🍌 每天学习就是在"剥香蕉"
- **月视图里**，达标的日期会显示 🍌，没达标的显示 ⏳，没学的显示 ·
- **24 小时时间线**里，你的学习时段是绿色高亮的，摸鱼时段灰暗
- **达到每日目标**会弹出弹窗「🎉 香蕉剥完啦！」配一句随机名言
- **名言库**选了 15 句经典，从孔子到村上春树都有

---

## 怎么用 🚀

1. 打开「剥香蕉」→ 点「选择」
2. 勾选你要用的学习 App（建议选 2 个分屏）
3. 点「开始专注」
4. 学习！不要切屏！切了赶紧切回来！
5. 学完点「结束学习」
6. 去「历史」看你的 🍌 日历
