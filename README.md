# 🍌 剥香蕉 — macOS 专注学习助手

> 选好学习 App → 开始专注 → 切屏就"断蕉"！  
> 每天学习达标，香蕉就剥完啦！🍌

---

## ✨ 这是什么？

**剥香蕉**是一款 macOS 专注学习工具。它的核心逻辑很简单：

1. **选择你要用的学习 App**（比如无边记 + Safari 分屏学习）
2. **点击「开始专注」**，屏幕自动暗化，只高亮学习窗口
3. **一旦你切到别的 App**，5 秒宽限期内切回来继续，超时则本次学习作废
4. **完成学习后**，时长自动记录，按日/周/月/年统计
5. **每天达成学习目标**，弹出庆祝动画 + 名人名言鼓励 🎉

---

## 🎯 核心功能

| 功能 | 说明 |
|------|------|
| 🖥️ **应用白名单** | 自动扫描 Mac 上所有 App，勾选学习用 App |
| 🌑 **屏幕暗化蒙层** | 非学习区域自动覆盖半透明暗色，只"照亮"学习窗口 |
| ⏱️ **精准计时** | 24 小时时间线，每小时学习情况一目了然 |
| 🚫 **切屏检测** | 切换到非白名单 App 立即触发宽限期 |
| ⚠️ **宽限期** | 默认 5 秒，可调 0~10 秒（0 = 切屏立刻作废） |
| 🍅 **番茄钟** | 可选 25+5 分钟自动循环，到点提醒休息 |
| 📊 **数据统计** | 日·周·月·年四维视图，月视图用 🍌 标记达成日 |
| 🔥 **连续打卡** | 追踪连续达标天数，保持 streak |
| 🔒 **锁屏暂停** | 锁屏/休眠自动暂停，解锁继续 |
| 🎉 **达标庆祝** | 达成每日目标弹出可爱动画 + 随机名言 |

## 🚀 安装

### 方式一：下载 DMG
1. 从 [Releases](../../releases) 下载最新 `剥香蕉-x.x.x.dmg`
2. 双击打开，拖动「剥香蕉」到 `Applications` 文件夹
3. 首次启动右键点击 → 打开（未签名应用需要确认）

### 方式二：自行构建
```bash
git clone https://github.com/你的用户名/剥香蕉.git
cd 剥香蕉
bash build.sh
open build/剥香蕉.app
```

**系统要求**：macOS 13.0+（Ventura 及以上）

## ⚙️ 权限说明

- **辅助功能权限**（可选）：用于屏幕暗化蒙层获取窗口位置。不开启也能正常使用切屏检测和计时功能。
- **通知权限**（可选）：用于番茄钟休息提醒。

---

## 🛠️ 技术栈

- SwiftUI + AppKit（macOS 原生）
- NSWorkspace 事件监听（切屏检测）
- Core Graphics 窗口捕获（屏幕暗化）
- UserDefaults 数据持久化
- Shell 脚本自动构建 + DMG 打包

---

## 📝 使用技巧

1. **分屏学习**：选 2 个 App（如 Notion + PDF Expert），用 macOS 分屏功能左右排布
2. **宽限设 0**：自制力强的可以在设置里把宽限期改为 0 秒，切屏立刻作废
3. **善用番茄钟**：25 分钟专注 + 5 分钟休息，4 轮后休息 15 分钟
4. **目标不要太大**：建议从每天 60 分钟开始，逐渐增加
